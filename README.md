🎯 Random Number Guessing Game (Python)

A beginner-friendly Python game where players try to guess a randomly generated number between 1 and 100. The game offers two difficulty levels, provides hints after every incorrect guess, and calculates the player's score based on the remaining attempts.

📌 Features
🎮 Two game modes:
Easy Mode – 10 attempts
Hard Mode – 5 attempts
🎲 Random number generation using Python's random module
📈 Higher/Lower hints after each guess
🏆 Score calculation based on remaining attempts
👤 Personalized gameplay with player name input
🖥️ Simple command-line interface
💡 Great beginner project for learning Python fundamentals
🛠️ Technologies Used
Python 3
Built-in random module

▶️ How to Run
Clone this repository:
git clone https://github.com/your-username/Random-Number-Guessing-Game-Python.git
Open the project folder:
cd Random-Number-Guessing-Game-Python
Run the program:
python random_number_guessing_game.py
🎮 Gameplay
Enter your name.
Select a difficulty:
Easy (10 attempts)
Hard (5 attempts)
Guess a number between 1 and 100.
The game will tell you whether to guess higher or lower.
Guess correctly before your attempts run out to win.
Your score is calculated from the number of attempts remaining.
